# 🏏 Professional Java Cricket Game

A console-based cricket simulation written in Java using OOP principles.

## Features
- Two teams with 5 players each
- Toss system
- Overs-based match
- Randomized outcomes per ball
- Ball-by-ball commentary
- Scoreboard and result saved to file

1. Compile:
   ```bash
   javac src/main/*.java
